Self-Supervised Domain Adaptation for Biosignals
Using Self-Supervised Learning to learn transferable and generalizable features 

This project is aimed at using self-supervised learning to build transferrable features for sleep-stage classification. The dataset and setup is borrowed from the NeurIPS BEETL Challenge. The image below shows the generic pipeline for self-supervised learning for biosignals.
Next, different proxy tasks are explored for learning joint representations of EEG and EOG signals from the Physionet dataset. The target supervised task is sleep-stage detection. The aim is to learn generalizable features which allow sleep-stage detection for subjects from different age categories (which are the different domains).

The image below shows example proxy tasks specific to the Physionet dataset which may allow building of generalizable representations.
